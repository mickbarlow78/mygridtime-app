# Phase 1: Foundation

## Objective
Scaffold the Next.js project with TypeScript, Tailwind CSS and Supabase. Create the full folder structure. Configure local Supabase development. Create environment variable template. Verify everything runs.

## Prerequisites
- Node.js 18+ installed
- Docker Desktop installed and running
- Supabase CLI installed (`npm install -g supabase`)
- Claude Code authenticated

## What to Build

### 1. Initialise Next.js Project
```bash
npx create-next-app@latest tkc-timetable --typescript --tailwind --app --src-dir --use-npm
```

### 3. Initialise Supabase
```bash
npx supabase init
npx supabase start
```
Verify local Supabase is running and note the output URLs and keys.

### 4. Create Supabase Client Files

**src/lib/supabase/client.ts** — Browser client using `@supabase/ssr`:
- Uses `createBrowserClient` from `@supabase/ssr`
- Reads `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY`

**src/lib/supabase/server.ts** — Server client:
- Uses `createServerClient` from `@supabase/ssr`
- Reads cookies from Next.js `cookies()` helper
- Used in Server Components and Route Handlers

**src/lib/supabase/middleware.ts** — Middleware helper:
- Refreshes auth session on every request
- Used by `src/middleware.ts`

### 5. Create Next.js Middleware
**src/middleware.ts**:
- Calls Supabase middleware helper to refresh session
- Matcher: exclude static files, _next, favicon

## Acceptance Criteria
- [ ] `npm run dev` starts without errors on localhost:3000
- [ ] `npm run build` completes without errors
- [ ] `npm run typecheck` passes with zero errors
- [ ] `npm run lint` passes
- [ ] `npx supabase start` runs local Supabase successfully
- [ ] `.env.local` is populated with local Supabase values
- [ ] `.env.local` is in `.gitignore`
- [ ] Every route renders its placeholder page without errors
- [ ] Folder structure matches the spec exactly
- [ ] `tsconfig.json` has `strict: true`

## Do NOT Build in This Phase

- Auth flow (Phase 2)
- Any UI beyond placeholders
- Any API logic
- Any Stripe/Twilio/AI integration
